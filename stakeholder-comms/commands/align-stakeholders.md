# /align-stakeholders
Plan a session that ends in real alignment.
## Steps
1. **Map the room** — Understand influence, interest, stance using `stakeholder-map` skill.
2. **Design the session** — Structure activities and a decision method using `alignment-workshop` skill.
3. **Prepare comms** — Draft the pre-read and follow-up using `executive-update` skill.
4. **Plan for impasse** — Define the escalation path using `escalation-plan` skill.
## Output
A facilitator-ready workshop plan with pre-read, agenda, decision method, and a fallback if the room can't agree.
Consider following up with `/report-status` to broadcast the decisions.
